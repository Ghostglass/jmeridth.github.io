Microsoft Web Application Stress (WAS)

Jan 23, 2007

Microsoft's Web Application Stress Tool provides an easy way to simulate large numbers of users against your Web application.

Web testing is divided into three main categories:

  • Performance testing
  • Stability or stress testing
  • Capacity planning
