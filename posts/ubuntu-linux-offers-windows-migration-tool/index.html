Ubuntu Linux offers Windows migration tool.

Mar 29, 2007

The newest version of Ubuntu 7.04 BETA is offering a new Windows migration tool.  According to the website, it will recognize and migrate "Internet Explorer favorites, Firefox favorites, desktop wallpaper, AOL IM contacts, and Yahoo IM contacts" during the install process.

I believe this is a huge step up to possibly convine a normal home user to switch to Linux.
